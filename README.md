# Personalized Learning Recommendation System

## Project Overview

This project recommends online courses based on a student's interests, skills, and learning goals using Machine Learning techniques.

## Features

* Course recommendation based on user interests
* TF-IDF Vectorization
* Cosine Similarity based recommendation engine
* Courses collected from multiple platforms:

  * Coursera
  * Udemy
  * edX
  * Skillshare

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Streamlit

## Files

* app.py → Main application
* final_courses.csv → Processed dataset
* tfidf.pkl → Saved TF-IDF model
* requirements.txt → Required libraries

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
